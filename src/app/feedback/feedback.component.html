Notice a bug, have a feature request, or just want to send some general feedback about the app? Please send all feedback to admin@houseofgate.com!

For Bug Reports

For bug reports, please include all (or as much as possible) of the following information:

  • Date bug happened
  • Your device, operating system, browser, and version. Ex: iPhone iOS 15.1, Safari browser
  • Where it happened
  • Detailed instructions of how to reproduce this bug
  • Any screenshots or recordings of this bug happening
