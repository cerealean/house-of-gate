Welcome to the House of Gate app!

The purpose of this application is to provide some useful tools to DMs and players. Please be aware that this app is currently in beta and features may change at any time! Any feedback can be directed to admin@houseofgate.com. You can follow changes on the changelog page.

Getting Started

To get started, go to Campaigns and create a new campaign! From there, you can create Encounters and save them to your campaign to use later.

There is also a monster search tool to let you search for monsters.

Optimization

This app is currently optimized for mobile devices and tablets. More desktop optimizations will come soon!

Saved Data

All campaign, encounter, and monster data is saved to your local device/machine. This means that if you erase local data, such as your browser cache, you might erase any saved campaign or encounter data you have in this app!

I will be creating ways for you to export this data and potentially store this data on a web server in the future so that there is less danger of you losing data.
