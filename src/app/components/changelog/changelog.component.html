Changes To The App

  • 2022-12-20
    • Upgraded dependencies to latest versions.
    • Fixed broken components in monster search and encounter generation.
    • Now displaying the calculation used on the distance calculator.
    • Squashed some bugs.
  • 2022-02-11
    • Added skills to character sheet.
    • Added the ability to edit most of the character sheet live while being on the character sheet.
  • 2022-02-04
    • Added a health modifier calculator to the character sheet. Simply click on the health bar on the character sheet to bring up the calculator!
    • The character sheet now has a max-width of 375px for better display.
    • The character sheet will now show the character's proficiency bonus.
    • Bugfix: Fixed bug that wouldn't show a newly added character in your characters list immediately after creating that character
  • 2022-01-29
    • Releasing the Characters Module
      • You can now create and add characters!
      • Characters can also be attached to Campaigns, if you'd like.
      • Character functionality is still basic and limited mostly to cosmetic use.
      • Improved character functionality is planned for the near future.
  • 2022-01-19
    • Added concentration column and filtering to spells
  • 2022-01-18
    • Fixed Distance Calculator to use numeric input on mobile devices and have better numeric validation
  • 2022-01-17
    • Added Distance Calculator to help players determine if they can hit an enemy on a different elevation than them (such as in the air).
  • 2022-01-11
    • Added sorting by column to monster and spell tables.
  • 2022-01-08
    • Fixed incorrect label on spell filter bar
    • Fixed bug where filtering spells to 0 results could break spell search.
    • Added feedback page for people to submit feedback.
  • 2022-01-07
    • Added Spell Search
    • Upgraded to Angular 13. Internet Explorer support fully removed.
    • Fixed bug where filtering monsters to 0 results could break monster search.
  • 2021-11-23
    • Added a home page
    • Added a changelog page
    • Removed the snackbar notice that popped up when you opened the application which stated that this app is in beta. It's no longer needed since the home page has that content.
  • 2021-11-22
    • Added the ability to delete encounters from My Encounters
